# Abhigyan — Resume

LaTeX resume auto-compiled to PDF via GitHub Actions and published to GitHub Pages.

## Files

- `abhigyan_resume.tex` — LaTeX source
- `resume.cls` — Document class (Trey Hunner's resume style)
- `.github/workflows/compile-and-deploy.yml` — CI/CD pipeline

## Usage

Push to `main` and GitHub Actions compiles the PDF automatically.

**Download:** [abster12.github.io/resume/resume.pdf](https://abster12.github.io/resume/resume.pdf)
